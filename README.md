# Финансовый трекер

Веб-приложение для учёта личных доходов и расходов с визуализацией статистики, фильтрацией и экспортом данных.

## Демо

[**Открыть демо-версию**](https://finance-tracker-artie.netlify.app/)

## О проекте

Приложение позволяет вести учёт финансовых операций, группировать их по категориям, отслеживать баланс, анализировать расходы и доходы за текущий месяц с помощью диаграмм, а также фильтровать и экспортировать данные.

## Используемые технологии

- **Фреймворк:** React 19
- **Сборщик:** Vite
- **Стилизация:** TailwindCSS
- **Диаграммы:** Recharts
- **Языки:** JavaScript (97.6%), HTML (2.1%), CSS (0.3%)

## Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/artvonbel/finance-tracker.git

# Перейти в папку проекта
cd finance-tracker

# Установить зависимости
npm install

# Запустить в режиме разработки
npm run dev

# Собрать для продакшена
npm run build

# Предпросмотр собранного проекта
npm run preview
```
